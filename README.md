### 📚 학습 가이드(필독)

- 학습하고 계신 매체에 맞춰 아래 **브랜치**를 선택해 진행해 주세요.
- 브랜치를 변경하면 각 환경에 최적화된 소스 코드를 확인하실 수 있습니다.

| 구분            | 추천 브랜치              | 비고                |
|:--------------|:--------------------|:------------------|
| **📖 도서 학습자** | `main` 또는 `insight` | 도서 출간물 기준 실습 코드   |
| **🎓 강의 학습자** | `inflearn/v1`       | 인프런 강의 커리큘럼 기준 코드 |
